INTERVIEW QUESTION

1. Difference between div and span

"div" tag stands for division tag and it is a block level element means it is bound to take the whole width of the screen. It divides the content into blocks
"span" tag is an inline element means it only takes the space that is required for the content . It is used to style a particular part of the content.

2. Difference between class and id

1. One element can have more than one class but one element can only have one id.
2. Id's are unique which means you can not give one id to multiple elements but classes are not unique.
3. When we style an element using id we use "#" but when we style using class we use "."

3. What are self closing tags

In HTML, self-closing tag is a type of tag that needs not be closely by closing tag, which means their is no seprate closing tag for it.

4. How many types of heading tags are there? How to decide which to use where

Their are 6 types of heading tags from h1 to h6

  • H1 tags are used when we want to give the main heading or title of the page
  • h2 and h3 are used to give the sub-heading in the page
  • H4,H5 and H6 are used to give the content of the page

5. Difference between block and inline

Block elements aquire the entire width of the screen.Example of block elements are div, p, h1 to h6, table , form etc.
Inline elements aquire only the width required for the content. Example of the inline elements are a, span, img, input, label etc.

6. Difference between em and strong

The strong tag should be used to indicate strong importance. It makes text look bold
the em tag should be used to make our text italic.

7. What are attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

8. Difference between html tags and elements

HTML tags are the starting and ending part of an HTML element. The begin with the "<" and end with ">".
HTML elements are the combination of the HTML tags and the content written between the tags.
